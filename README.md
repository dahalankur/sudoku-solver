# sudoku-solver
A C++ sudoku solver
By Ankur Dahal

This is a sudoku solver written in C++. To compile and run the program, type
            * make sudoku *
            * ./sudoku    *
If you want to provide the board input to the program through a text file, use 
            * ./sudoku filename *
